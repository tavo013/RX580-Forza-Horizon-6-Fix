# Fix para RX580 - Forza Horizon 6
FH201 &amp; FH205 Fix

Basicamente o jogo foi lançado para a versão do DirecX12.1 e as placas mais antigas como a RX580 só possuem suporte ao DirectX 12.0

Este e um Fix Experimental para as placas de vídeo RX580 AMD | 8GB VRAM | 2304SP/2048SP
Esse Fix não funciona para a versão atual do Driver disponível para a RX580, pois basicamente o último Driver para esta placa limitam ela.

# Passo a Passo

1. Faca o Download da Pasta BIN la vai estar tudo que é necessário nesse fix.
2. Extraia a DLL para a pasta do jogo
3. Instale o Driver 23.10.01.14 para a RX580
4. Abra as Propriedades do Executável do jogo e ative a Compatibilidade Windows 8 aplique e jogue o jogo.

Pode haver artefatos e bugs durante a gameplay justamente por conta do DirecX12.1 do jogo.
